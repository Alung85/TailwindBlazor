TailwindBlazor

Zero-config Tailwind CSS v4 integration for Blazor.

Auto-downloads the CLI. Compiles CSS at build time. Runs watch mode in dev. One package, one line of code.

Quick Start

  1. Install: dotnet add package TailwindBlazor
  2. Create Styles/app.css with: @import "tailwindcss";
  3. Add builder.UseTailwind(); to Program.cs

Features

  • Auto CLI Download — detects OS/arch, caches binary
  • MSBuild Integration — build-time CSS generation
  • Watch Mode — instant CSS rebuilds in development
  • Fully Configurable — MSBuild, C#, or appsettings.json
